# TwerkOS

TwerkOS is built for users who want a faster, cleaner, and more optimized system setup without manually changing dozens of settings.

### Important 
* Some optimizations may require administrative privileges and a system restart to fully take effect.

### Installation & Usage

* Download the TweakOS package.
* Extract the files into a preferred directory.
* Run the executable or script as administrator.
* Follow the on-screen instructions.
* Restart the system if required to apply all changes.
